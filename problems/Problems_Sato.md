## 弾丸

### ジャンル

ループ

### 問題

厚さT<sub>i</sub>の鉄板のリストが与えられる。
貫通力Pの砲弾を打ち込んだ時に何枚目まで貫通できるか求めよ。
なお、厚さT<sub>i</sub>の鉄板を貫通すると、砲弾の貫通力はP-T<sub>i</sub>となる。

### 制約

* 1 <= N <= 100
* 1 <= T<sub>i</sub> <= 100
* 1 <= P <= 1000


## 斜め読み

### ジャンル

文字列、配列

### 問題

**と** にかく臨機応変に隣人

と **も** 善き関係を築くべく

から **だ** と心を健全に鍛え

たびだ **ち** のその日まで、

共に想い **募** らせ励まし合い

皆の信望を **集** める人間になろう！ 

### 制約

* 1 <= H, W <= 100
* H = W
* 英字のみ出現

## 極大点

### ジャンル

配列内の比較

### 問題

数列aが与えらる。
a<sub>i-1</sub> < a<sub>i</sub> > a<sub>i+1</sub>となるiの個数を求めよ。
両端(i = 0, i = N - 1)は答えに含めない。

### 制約

* 1 <= N <= 1000
* 1 <= a<sub>i</sub> <= 10
