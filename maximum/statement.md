# 極大点

## 問題

数列a<sub>i</sub> (i=1,2,...,N)が与えらる。
a<sub>i-1</sub> < a<sub>i</sub> > a<sub>i+1</sub>となるiの個数を求めよ。
両端(i = 0, i = N - 1)は答えに含めない。

## 入力

複数のデータセットから成る。
1つのデータセットは以下の通り。

<pre>
N
a<sub>1</sub> a<sub>2</sub> ... a<sub>N</sub>
</pre>

入力の最後は"0"。

## 制約

* 1 <= N <= 1000
* 1 <= a<sub>i</sub> <= 10

## 出力

それぞれのデータセットに対して、極大点の個数を1行で出力せよ。
